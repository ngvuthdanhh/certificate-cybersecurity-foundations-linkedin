# Module 4 – Security Controls

## Types of Controls
- **Administrative** – Policies, training, procedures
- **Technical** – Firewalls, antivirus, access control
- **Physical** – Locks, cameras, facility security

## Access Management
- Authentication (proving identity)
- Authorization (what you can do)
- Accounting (logging & auditing)

## Defense in Depth
Layered approach: multiple safeguards prevent single point of failure.

## Key Insight
Security controls reduce **risk exposure** and improve resilience.
